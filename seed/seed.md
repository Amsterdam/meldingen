## Seed files

This folder stores the seed files used to initialize the database with default or sample data. 
Seed files are stored as JSON and are imported in the meldingen-entrypoint.sh script through Typer Commands (in the /commands folder). 

### Currently supported seed files

| Seed File Name       | Command file       | Typer Command                       | 
|----------------------|--------------------|-------------------------------------| 
| classifications.json | classifications.py |  classifications seed |
